# 🚀 노션 포트폴리오 SEO (검색엔진 최적화) 작성 가이드

노션에 포트폴리오를 작성하실 때 아래 가이드라인을 지켜주시면, 구글 로봇이 내용을 훨씬 더 잘 이해하고 검색 결과 상단에 노출될 확률이 비약적으로 높아집니다!

---

## 1. 제목(Name)은 '키워드'를 포함해서 구체적으로!
구글은 웹페이지의 제목을 가장 중요하게 생각합니다. 단순히 클라이언트 이름만 적는 것보다, **고객이 검색할 만한 키워드**를 함께 적어주세요.

> [!WARNING] 아쉬운 예시
> ❌ A기업
> ❌ B성형외과

> [!TIP] 좋은 예시
> ✅ A기업 엔터프라이즈 반응형 웹사이트 제작
> ✅ B성형외과 브랜딩 및 하이엔드 홈페이지 구축

## 2. 요약 설명(Description)은 매력적으로!
Description 속성은 구글 검색 결과에서 제목 밑에 뜨는 **미리보기 텍스트(Meta Description)** 로 활용됩니다. 
사람들이 클릭하고 싶도록 1~2줄로 핵심을 요약하고, 관련 키워드를 자연스럽게 포함해 주세요.

> **예시:** "강남 B성형외과의 브랜드 가치를 높이는 프리미엄 웹사이트 제작 사례입니다. 전환율을 극대화하는 UI/UX 설계와 예약 시스템 연동을 진행했습니다."

## 3. 본문(블록)은 '텍스트' 중심으로 꼼꼼하게!
구글 로봇은 아직 이미지를 눈으로 볼 수 없습니다. **글자(텍스트)** 를 읽고 사이트를 평가합니다. 통이미지로만 포트폴리오를 구성하면 구글 입장에서는 텅 빈 페이지로 보입니다.

* **프로젝트 배경 / 해결 과제 / 성과** 등을 상세하게 글로 적어주세요.
* 글의 구조를 잡을 때 노션의 **제목1(H1), 제목2(H2), 제목3(H3)** 기능을 적극적으로 활용하세요. 구글은 제목 태그를 통해 글의 맥락을 이해합니다.

## 4. 이미지 업로드 전 파일명 변경 & 캡션 달기
이미지는 검색 상위 노출에 큰 영향을 미칩니다. (구글 이미지 검색에서도 유입됩니다!)

* **파일명 최적화:** 카카오톡에서 다운받은 `KakaoTalk_20231201...jpg` 형태의 파일명을 그대로 올리지 마세요. 올리기 전에 파일명을 `a-company-website-main.png` 처럼 **영문 키워드**로 변경해서 올려주세요.
* **캡션 달기:** 노션에서 이미지를 넣은 후, 이미지 바로 아래에 **'캡션(설명)'**을 달아주시면 구글이 이미지를 이해하는 데 큰 도움이 됩니다. (예: "A기업 메인 페이지 디자인")

## 5. 라이브 링크(URL) 삽입
실제 제작된 웹사이트가 퍼블릭하게 열려있다면, 포트폴리오 본문 하단에 실제 웹사이트로 이동하는 링크를 넣어주세요. 서로 연결된 웹사이트들은 구글이 더 신뢰할 수 있는 정보로 평가합니다.
