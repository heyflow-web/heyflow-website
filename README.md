# heyflow. Official Website Deep-Dive Project Blueprint

## 1. Project Essence & Philosophy
"우리는 본질만 남기고, 시선의 흐름을 디자인합니다."

- **Project Name:** heyflow. (헤이플로우 공식 웹사이트 구축)
- **Target Core:** 일반 기업 브랜드 디렉터, IT 스타트업, 하이엔드 F&B 및 커머스 (B2B 수주 목적)
- **Infrastructure:** Vercel (초고속 성능 및 부드러운 인터랙션 인터페이스 확보)
- **Design Core:** 화려한 기교나 비주얼 소음(Noise)을 극도로 배제한 프리미엄 미니멀리즘(Premium Minimalism). 과감한 여백, 타이포그래피의 세련된 대비, 그리고 자연스러운 시선의 이동(Visual Flow) 설계.

## 3. Contact Form Architecture
고객의 프로젝트 의뢰를 수집하는 프리미엄 다단계 문의 폼입니다.

- **Flow:** Typeform 스타일의 One-step-at-a-time 모달 인터페이스 적용
- **Backend Storage:** 구글 Apps Script 연동을 통한 Google Spreadsheet 자동 적재 및 이메일(travel202@naver.com) 실시간 알림
- **Steps:**
  1. 기본 정보 (이름/브랜드명)
  2. 본질적 문제 및 목표
  3. 예산 범위 (100만원 이하 / 100~300만원 / 300만원 이상)
  4. 연락처 (전화번호/이메일)
  5. 제출 완료 (Outro Fade-in)
